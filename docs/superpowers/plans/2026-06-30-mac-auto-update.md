# Mac Auto Update Implementation Plan

> Goal: keep first install as a DMG, then let packaged macOS users update inside Mineradio with one click.

## Architecture

Use `electron-updater` in the main process as the trusted updater. The renderer keeps the existing update modal and progress UI, but it asks the main process to check, download, and install updates.

Release source is GitHub Releases at `IFPrice/Mineradio`. A release must include:

- `Mineradio-<version>-mac-arm64.dmg` for first-time installs.
- `Mineradio-<version>-mac-arm64.zip` for automatic updates.
- `latest-mac.yml` generated by `electron-builder`.

The current server-side DMG download remains as a fallback for development builds or updater failures. The existing resource patch path stays lower priority because patching app resources can conflict with macOS signing.

## Tasks

1. Add updater dependency and release metadata.
   - Add `electron-updater` to production dependencies.
   - Change `build.publish.owner` and `mineradio.update.owner` from `XxHuberrr` to `IFPrice`.
   - Keep DMG and ZIP mac targets enabled.

2. Add main-process updater bridge.
   - Import `autoUpdater` from `electron-updater`.
   - Set GitHub feed to `IFPrice/Mineradio`.
   - Disable automatic download and install-on-quit.
   - Add IPC handlers:
     - `mineradio-auto-update-check`
     - `mineradio-auto-update-download`
     - `mineradio-auto-update-install`
   - Broadcast updater states to the renderer:
     - `checking`, `available`, `not-available`, `downloading`, `downloaded`, `error`.

3. Expose safe preload methods.
   - Add `checkAppUpdate`, `downloadAppUpdate`, `installAppUpdate`.
   - Add `onAppUpdateState` with listener cleanup.

4. Wire the existing update modal.
   - Prefer updater check result when available.
   - Button flow:
     - available -> download update
     - downloading -> disabled/progress only
     - downloaded -> restart and install
   - Fall back to existing DMG download if the app is not packaged or updater is unavailable.

5. Verify.
   - Static tests assert updater wiring and release owner.
   - Run `node --test test/*.test.js`.
   - Sync to `local/`.
   - Build unsigned arm64 DMG/ZIP with `CSC_IDENTITY_AUTO_DISCOVERY=false npm run build:mac:arm64`.
   - Verify the DMG with `hdiutil verify`.
